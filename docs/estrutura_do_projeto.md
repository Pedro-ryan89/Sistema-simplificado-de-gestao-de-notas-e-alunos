# Estrutura do Projeto

Este projeto sera organizado de forma modular para facilitar leitura, manutencao e explicacao.

## Arquivos principais

- `main.py`: ponto de entrada do sistema.
- `menu.py`: versao inicial/rascunho atual do menu.
- `sistema/menus.py`: menus e submenus.
- `sistema/alunos.py`: cadastro, listagem, edicao, exclusao e media.
- `sistema/consultas.py`: busca e filtros.
- `sistema/estatisticas.py`: estatisticas da turma.
- `sistema/armazenamento.py`: salvar e carregar dados.

## Pastas

- `dados/`: arquivos salvos pelo sistema.
- `docs/`: anotacoes, explicacao e testes manuais.
- `rascunhos/`: ideias e testes temporarios.
